Este proyecto fue el trabajo realizado para acabar el curso de Administración de Bases de Datos. En este trabajo se tocan los siguientes puntos:
-Creación de PDB mediante dbca
-Creación de un usuario administrador para la pdb
-Perfil de recursos
-Asignación de privilegios
-Consultas a la base de datos
-Creación de tablas, inserciones, merge

Como parte de este trabajo podemos encontrar un pdf explicativo y 2 documentos sql.
